# ApiFood 🍎

### Pagina Web creada para hacer consultas de comidas saludables, donde podes filtrar, ordenar y realizar busquedas por nombre, podes ver el detalle de una comida seleccionada, tambien podes crear una receta y paginado.

### 1. Clona el repositorio

### 2. Copia y pega los siguientes comandos en la consola de Visual Studio

### 3. `npm install`

### 4. `npm start`

### 5. Abri http://localhost:3000 en tu navegador
